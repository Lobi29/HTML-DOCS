## Attributes, Headings and Paragraphs

<br />
<br />

<h3>Attributes : notes</h3>
<p>HTML attributes provide additional information about HTML elements.</p>
<ul>
    <li>All HTML elements can have attributes</li>
    <li>The href attribute of <strong><a></strong> specifies the URL of the page the link goes to</li>
    <li>The src attribute of <strong><img></strong> specifies the path to the image to be displayed</li>
    <li>The 'style' attribute is used to add styles to an element, such as color, font, size, and more</li>
    <li>The 'lang' attribute of the <strong><html></strong> tag declares the language of the Web page</li>
    <li>The 'title' attribute defines some extra information about an element</li>
</ul>


<br />
<br />


<h3>Headings : notes</h3>
<p>HTML headings are titles or subtitles that you want to display on a webpage</p>
<ul>
    <li>It is range from h1 to h6</li>
    <li><h1> defines the most important heading. <h6> defines the least important heading.</li>
</ul>

<h3>Paragraphs : notes</h3>
<p>A paragraph always starts on a new line, and is usually a block of text</p>
